# jacgunsalus.github.io
created during the summer of 2018
calculates recommended Tylenol dosages based on age and weight (dosage information was found on the official Tylenol website)
I noticed that it was pretty tricky to figure out effective tylenol dosages for children, especially kids under the age of 6, so I compiled all of the guidelines based on age and weight into my code and built a simple dosage calculator. 
first project made using CSS, Javascript, and HTML
